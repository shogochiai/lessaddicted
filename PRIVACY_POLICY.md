# Privacy Policy for LessAddicted Chrome Extension

This Privacy Policy describes how the LessAddicted Chrome Extension operates regarding user data.

## 1. Data Collection
The LessAddicted Chrome Extension does not collect, transmit, or store any personal data from its users. Our extension's sole function is to automate the display of your chosen custom list on X.com (formerly Twitter).

## 2. Information Stored Locally
The extension only stores the following non-personal preferences directly on your local machine, within your Chrome browser's synchronized storage. This data is used exclusively by the extension to provide its intended functionality and is not accessible by, or shared with, the developer or any third parties:

The name of the custom X.com list you have set as your default.

An internal state indicator (isSettingMode) used to manage the one-time setup process.

Your preferred language for the extension's user interface.

## 3. No External Communication
LessAddicted does not communicate with any external servers. All operations and data storage are performed entirely locally within your browser.

## 4. No Data Sharing or Sale
No user data, personal or otherwise, is ever sold, rented, shared, or transferred to third parties for any purpose.

## 5. Changes to This Privacy Policy
Any updates to this Privacy Policy will be reflected in new versions of the extension published on the Chrome Web Store.

## 6. Open Source Transparency
For complete transparency and to verify the extension's operation, the full source code is publicly available on GitHub at: https://github.com/shogochiai/lessaddicted

## 7. Contact Us
For any questions regarding this Privacy Policy or the LessAddicted extension, please refer to our GitHub repository.
